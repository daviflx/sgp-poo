# 🏥 SGP-POO — Sistema de Gestão de Pacientes

Projeto desenvolvido em **Java** com foco na aplicação prática dos conceitos de **Programação Orientada a Objetos (POO)**, como:

- ✅ Abstração
- ✅ Encapsulamento
- ✅ Herança
- ✅ Polimorfismo

O sistema simula um ambiente de gestão de usuários e suas demandas, permitindo o cadastro e manipulação de informações de forma estruturada e organizada.
